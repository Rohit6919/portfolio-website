React Portfolio Website
This is the frontend for my personal portfolio website, built using React, Vite, and Tailwind CSS. The site is designed to display information about my skills, projects, and provide a contact form for users to reach out to me.

About
This project is a personal portfolio website built with modern web technologies. It features a simple, yet stylish design to showcase my professional background, skills, and projects.

Key Features:
Responsive Layout: Optimized for mobile and desktop views.

Project Gallery: Showcases my projects and provides detailed information.

Contact Form: Allows users to send messages directly from the website.

Technologies Used
React: JavaScript library for building user interfaces.

Vite: Next-generation build tool for fast and optimized React apps.

Tailwind CSS: A utility-first CSS framework for rapid UI development.

Node.js (Backend): Server-side technology for API handling.

Express.js (Backend): Web framework for Node.js for building the backend.

MongoDB: NoSQL database for storing contact form submissions.
