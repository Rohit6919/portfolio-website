# Portfolio Website

## Description

This project is a personal portfolio website built with React.js (frontend) and Node.js/Express (backend). The website showcases the various projects, skills, and achievements of the developer. The backend is powered by MongoDB for data storage and provides APIs for saving and retrieving contact data.

## Features

- **Frontend**: Built with React.js, showcasing portfolio projects and achievements.
- **Backend**: Built with Node.js and Express, providing APIs for storing and retrieving contact data.
- **MongoDB**: Used to store contact messages from visitors.
- **Deployment**: Both frontend and backend are deployed separately on Vercel.

## Tech Stack

- **Frontend**: React.js, Vite (build tool)
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Vercel
